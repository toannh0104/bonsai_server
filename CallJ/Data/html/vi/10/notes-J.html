Key Forms:

The following examples show the key forms to be practised in this lesson.

1Polite vs Plain:
父は新聞を読んでいますMy father is reading the newspaper.
父は新聞を読んでいるMy father is reading the newspaper.
2Question vs Statement:
サリーさんは皿を洗っていますかIs Sally washing the dishes?
サリーさんは皿を洗っていますSally is washing the dishes.

The sentences you create in this lesson should all be similar to the forms shown above. The grammar required to form these sentences is covered in the text below.



Grammar Point 1:

Sentences which express a state or ongoing action are formed by using the "te" form of the verb plus the auxiliary verb "iru". This pattern corresponds closely to the combination of "be" plus a gerund in English. (For example, "I am reading.") Unlike sentences which express existence using only the verb "iru", in this pattern "iru" is used for both people and inanimate objects. The polite form of this pattern is formed by using "imasu", the polite form of "iru".

Plain
ジョンさんは喫茶店でコーヒーを飲んでいる - "John is drinking coffee in a coffee shop." (plain)
妹は本を読んでいる - "My younger sister is reading a book." (plain)
Polite
田中さんは学校でサリーさんを待っています - "Tanaka is waiting for Sally at school." (polite)
兄はサムさんと話しています - "My older brother is talking to Sam." (polite)

Adding the question marker "ka" changes the statement into a question.

Question, Polite
鈴木さんはテレビを見ていますか - "Is Suzuki watching television?"
田中さんは魚を食べていますか - "Is Tanaka eating fish?"
